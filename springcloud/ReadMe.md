### 规则 
```新建的module以 9000~40000之间
    例如  110Consumer port 11100   端口最后一位为调整端口防止被占
         111Consumer port 11110
         120Provider port 11200  
         121Provider port 11210  

⼀般⽤到的是1到65535,其中0不使⽤。
1-1023为知名端⼝（Well-Known Ports）。
1024-49151为⽤户端⼝（Registered ports）。
49152-65535称为动态端⼝（Dynamic Ports）。
知名端⼝即众所周知的端⼝号，范围从0到1023，这些端⼝号⼀般固定分配给⼀些服务。
⽤户端⼝由IANA负责分配，需要⾛申请流程，申请⼿续相对系统端⼝来说不那么严格。
动态端⼝的范围从49152到65535，这些端⼝号⼀般不固定分配给某个服务，也就是说许多服务都可以使⽤这些端⼝。
```
